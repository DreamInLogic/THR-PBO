📚 Sistem Pengumpulan Tugas (Java PBO)

Project ini merupakan implementasi sistem pengumpulan tugas berbasis Java sebagai tugas mata kuliah Pemrograman Berorientasi Objek (PBO).

📂 Struktur Project
.
├── tanpa-db/        # Versi tanpa database (array / OOP dasar)
├── dengan-db/       # Versi menggunakan database MySQL
└── README.md

🗂️ Versi Project
🔹 1. tanpa-db

Menggunakan konsep OOP dasar tanpa database:

Data disimpan di array

Cocok untuk pembelajaran dasar Java

🔹 2. dengan-db

Menggunakan database MySQL:

Data tersimpan permanen
